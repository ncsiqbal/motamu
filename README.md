## 📱 Project Overview

**Mo-Tamu** is an Android-based guest management application that leverages **face recognition technology** to enhance residential security systems.

The application enables real-time guest identification, logging, and monitoring, and is integrated with an emergency response mechanism connected to a web-based dashboard.

---

## 🎯 Project Objectives

- Strengthen residential security using face recognition
- Automate guest registration and identification
- Enable real-time monitoring and reporting
- Provide emergency response integration

---

## ✨ Key Features (High-Level)

- Guest registration (permanent & non-permanent)
- Face detection and recognition
- Real-time guest logging
- Emergency button linked to monitoring dashboard
- Daily, weekly, and monthly visit statistics
- Integration with web monitoring system

---

## 🛠 Technology Stack

### Mobile
- **Platform:** Android
- **Language:** Kotlin
- **IDE:** Android Studio

### Backend
- **Language:** Python
- **Framework:** Flask
- **Database:** Firebase Realtime Database

### Face Recognition
- Haar Cascade (face detection)
- Local Binary Pattern Histogram (LBPH) for face identification
- OpenCV, NumPy, Scikit-Learn

---

## 🧱 System Architecture (High-Level)

- Android application handles image capture and user interaction
- Backend API processes face recognition requests
- Firebase stores guest and visit data
- Web dashboard displays monitoring and analytics

> Detailed implementation is excluded for confidentiality.
